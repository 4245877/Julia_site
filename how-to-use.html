Лаунчер

Лаунчер Julia — это основное приложение для управления ИИ. На главной форме лаунчера расположены пять кнопок:

  1. Запуск или инсталляция — запускает ИИ, если он установлен, или начинает процесс установки.
  2. Настройки — открывает меню для настройки параметров ИИ.
  3. Кастомизация внешности — позволяет изменить внешний вид ИИ.
  4. Переход на сайт — открывает официальный сайт Julia в браузере.
  5. Расположение ядра ИИ — открывает проводник с директорией, где находится ядро ИИ.

Настройки лаунчера

Лаунчер Julia предоставляет ряд настроек для тонкой конфигурации работы приложения. Ниже приведены основные разделы настроек и их параметры:

Раздел "File and Folder Paths"

В этом разделе указываются пути к различным директориям и файлам, используемым лаунчером:

  • Installation directory — путь к директории установки ИИ.
  • Log folder — путь к папке с логами приложения.
  • External modules paths — пути к внешним модулям, которые могут быть подключены к ИИ.
  • Change directory — возможность изменить рабочую директорию лаунчера.

Раздел "Resource Allocation"

Настройки, связанные с распределением системных ресурсов для работы ИИ:

  • RAM usage — ограничение использования оперативной памяти.
  • CPU load limitation — ограничение нагрузки на процессор.
  • Number of CPU cores to use — количество ядер процессора, используемых ИИ.
  • CPU load percentage — процент загрузки процессора.
  • Enable/disable GPU acceleration — включение/выключение ускорения с помощью GPU.
  • Limit GPU usage (e.g., by percentage) — ограничение использования GPU (например, в процентах).
  • Select a specific GPU if multiple are available — выбор конкретного GPU, если их несколько.
  • Maximum download/upload speed — максимальная скорость загрузки/выгрузки данных.

Раздел "Startup and Update Settings"

Настройки, касающиеся запуска и обновления лаунчера:

  • Auto-Start — автоматический запуск лаунчера при старте системы.
  • Updates Preferences — предпочтения по обновлениям:
    • Automatic updates — автоматические обновления.
    • Check for updates on startup — проверка обновлений при запуске.
    • Manual updates — ручные обновления.
    • Choose update branch (Stable, Beta, Nightly) — выбор ветки обновлений (стабильная, бета, ночная сборка).

Раздел "Logging and Debugging"

Настройки логирования и отладки:

  • Log detail level — уровень детализации логов:
    • Errors — ошибки.
    • Warnings — предупреждения.
    • Informational messages — информационные сообщения.
    • Debugging — отладочная информация.
  • Log format (txt, json, etc) — формат логов (текстовый, JSON и т.д.).
  • Log Retention — политика хранения логов.

Раздел "UI and User Preferences"

Настройки интерфейса и пользовательских предпочтений:

  • Theme — выбор темы оформления (светлая, темная и т.д.).
  • Language selection — выбор языка интерфейса.
  • Hotkeys — настройка горячих клавиш.

Раздел "Security"

Настройки безопасности:

  • Log and Data encryption (if needed) — шифрование логов и данных (при необходимости).
  • Settings protection with a Password — защита настроек паролем (для предотвращения случайных изменений).

Процесс установки и настройки

  1. Скачайте лаунчер с официального сайта.
  2. Запустите лаунчер и нажмите кнопку "Запуск или инсталляция" для установки ИИ.
  3. Перейдите в "Настройки" для конфигурации параметров ИИ.
  4. Используйте "Кастомизация внешности" для настройки внешнего вида.
  5. Нажмите "Запуск или инсталляция" для запуска ИИ после настройки.
